# 🍳 AI Cooking To-Do Planner

An AI-inspired meal planning web application that helps users create a personalized daily cooking plan based on their budget, dietary preferences, available ingredients, and cooking time.

Built using **HTML**, **CSS**, **Bootstrap 5**, and **Vanilla JavaScript**.

---

## 🚀 Features

### 🥗 Smart Meal Planning
Generate a complete daily meal plan including:

- Breakfast
- Lunch
- Dinner

Each meal includes:

- Dish Name
- Estimated Cost
- Cooking Time

---

### 🛒 Grocery List Generator

Automatically identifies missing ingredients by comparing:

- User's available ingredients
- Ingredients required for the generated meal plan

Creates a shopping-ready grocery list.

---

### 🔄 Ingredient Substitutions

Provides healthier or alternative ingredient suggestions.

Examples:

| Original | Substitute |
|-----------|------------|
| Milk | Almond Milk |
| Paneer | Tofu |
| Chicken | Soya Chunks |
| Butter | Olive Oil |

---

### 💰 Budget Feasibility Analysis

Calculates:

- Total Meal Cost
- Budget Usage
- Remaining Budget

Budget Status:

- ✅ Within Budget
- ⚠️ Near Budget
- ❌ Over Budget

---

### ✅ Cooking To-Do Checklist

Generates a simple cooking workflow:

- Wash Ingredients
- Prepare Ingredients
- Cook Meals
- Serve Food

Users can track progress with checkboxes.

---

### 🎲 Random Meal Suggestions

Provides quick inspiration when users are unsure what to cook.

---

### 📄 Export & Print

Supports:

- Download Meal Plan
- Print Meal Plan

---
